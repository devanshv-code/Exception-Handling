# Exception-Handling
## What is Exception ? 
An exception is an event that happens during a program's execution and disturb the normal flow of the program.
## Why it is Dangerous ? 
1.  Lead to sudden termination of the program.

2.  Can block the application.

3.  Data loss problem.

4.  Corrupt data files.
